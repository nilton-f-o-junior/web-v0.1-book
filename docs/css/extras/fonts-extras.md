# Fonts Extras

| Propriedade | O que faz | Valores Comuns / Exemplos |
|---|---|---|
| `font-display` | Controla a exibição enquanto carrega | `swap`, `block`, `fallback`, `optional` |
| `font-variant` | Ativa variações | `normal`, `small-caps` |
| `font-stretch` | Estica ou comprime a largura | `condensed`, `expanded`, `normal`, `75%` |
| `font-kerning` | Ajusta o espaço entre pares de letras específicos | `auto`, `normal`, `none` |
| `font-optical-sizing` | Otimiza o desenho da letra para o tamanho atual. | `auto`, `none` |
| `font-size-adjust` | Mantém o tamanho visual se a fonte principal falhar. | `0.5`, `none`, `from-font` |
| `font-synthesis` | Permite/proíbe o navegador crie negrito/itálico falso. | `none`, `weight`, `style`, `small-caps` |
| `font-feature-settings` | OpenType (ligaduras). | `"liga" 1`, `"smcp" 1`, `"frac" 1` |
| `font-variation-settings` | Controle total de eixos  | `'wght' 550`, `'wdth' 125` |
| `font` | Definições em 1 linha | `italic bold 16px/1.5 "Arial"` |
