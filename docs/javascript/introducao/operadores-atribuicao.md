# Operadores de Atribuição

`+=`

```js
// soma
let a = 12;
console.log(a += 1);
```

`-=`

```js
// subtracão
let a = 12;
console.log(a -= 1);
```

`*=`

```js
// multiplicacão
let a = 12;
console.log(a *= 2);
```

`/=`

```js
// divisão
let a = 12;
console.log(a /= 2);

```

`%=`

```js
// resto
let a = 12;
console.log(a %= 1);
```
