# Font

| Propriedade | O que faz | Valores Comuns / Exemplos |
|---|---|---|
| `font-family` | Família da fonte | `'Roboto'`, `Arial`, `sans-serif`, `serif` |
| `font-size` | Tamanho do texto | `16px`, `1rem`, `1.2em`, `100%` |
| `font-weight` | Espessura ou "peso" da fonte | `normal`, `bold`, `400`, `700`, `900` |
| `font-style` | Inclinação da fonte | `normal`, `italic`, `oblique` |

`html`

```html
<h1>Programação</h1>
<p>Este é um parágrafo de exemplo.</p>
```

`css`

```css
h1 {
    font-family: 'Montserrat', sans-serif;
    font-size: 3rem;
    font-weight: 900;
}

p {
    font-family: 'Montserrat', sans-serif;
    font-size: 3rem;
    font-weight: 900;
}
```
