# Tags

`h1-h6`

```html
<h1>h1</h1>
<h2>h2</h2>
<h3>h3</h3>
<h4>h4</h4>
<h5>h5</h5>
<h6>h6</h6>
```

`p`

```html
<p>p</p>
```

`br`

```html
<p>p <br> p</p>
```

`strong`

```html
<p><strong>negrito</strong></p>
```

`em`

```html
<p><em>itálico</em></p>
```

`mark`

```html
<p><mark>marca texto</mark></p>
```

`ins`

```html
<p><ins>sublinhado</ins></p>
```

`del`

```html
<p><del>taxado/riscado</del></p>
```

`sobrescrito`

```html
<p>x<sup>2</sup></p>
```

`subscrito`

```html
<p>H<sub>2</sub>O</p>
```

`cite`

```html
<p><cite>segundo ...</cite></p>
```

`blockquote`

```html
<blockquote cite="link do livro/artigo">
    Segundo ...
</blockquote>
```

`abbr`

```html
<abbr title="Hypertext ...">HTML - Abreviação</abbr>
```

`hr`

```html
<hr>
```
